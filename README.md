ssdl
----------
Download SlideShare slides as PDFs

# Be Warned!
It's horribly dependent on the structure of pages and may not work in the near future

# How to Use
```shell
pip install -r requirements.txt
python3 ssdl.py <URL of slide here>
```

# Requirements
 - Python 3.x
 - ImageMagick
 - beautifulsoup4
 - tqdm
